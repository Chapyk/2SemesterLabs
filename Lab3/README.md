# Memoization Function

## Description
This project implements a memoization function that caches results of a function to avoid repeated calculations.

## Features
- Caches function results
- Uses Map for fast lookup
- Supports cache size limit
- LRU (Least Recently Used) eviction strategy

## How to run
1. Make sure you have Node.js installed
2. Run:
   node test.js

## Example
The test demonstrates:
- caching results
- avoiding recalculation
- removing old cache entries when limit is reached
